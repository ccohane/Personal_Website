Overwatch League CryptoBets

About OWL CryptoBets

OWL CryptoBets is a decentralized application which allows it's users to bet on OWL Stage Playoff games with ether. I developed a smart contract in Solidity and tested it using Truffle and the Remix IDE. The front end was create using React framework. It is currently deployed to the Ropsten Test Network where you can place bets on who will come in first place and second place. Users recieve ERC721 Tokens for their bets and will be able to recieve prize money if they have a correct token after stage results are set.
